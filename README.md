# 🎮 Hangman Game

A professional text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

---

## 📌 Features

✅ Random word selection

✅ Interactive console interface

✅ Visual Hangman stages

✅ Input validation

✅ Maximum 6 incorrect guesses

✅ Win/Lose detection

---

## 🛠️ Technologies Used

- Python 3
- Random module
- Loops
- Functions
- Lists
- Strings

## 🚀 How to Run

Clone the repository

```
git clone https://github.com/samyakjainx/CodeAlpha_Hangman_Game.git
```

Move into the project

```
cd CodeAlpha_Hangman_Game
```

Run

```
python Hangman_Shell.py
```
